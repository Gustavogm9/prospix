## Regras de atualizacao

- Atualizar ao criar issue, abrir PR, bloquear, transferir dono, receber gate ou concluir.
- Nunca apagar linha concluida; mudar status para `Concluido`.
- Se virar risco tecnico, criar ou vincular `AUD-*` em [matriz-achados.md](matriz-achados.md).
- Se trocar de agente, registrar handoff usando [template-handoff.md](template-handoff.md).
